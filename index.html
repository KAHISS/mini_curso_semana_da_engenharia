Os Fundamentos: Lógica de Programação

Antes de analisarmos dados, precisamos aprender a "pensar como um computador". A lógica de programação é a base que nos permite criar instruções claras e eficientes para resolver problemas. Nesta seção, exploraremos os conceitos essenciais que formam o alicerce de qualquer programa ou análise de dados.

Variáveis

Espaços na memória para armazenar dados (números, textos, etc.).

idade = 30
nome = "Maria"

Tipos de Dados

Categorias de informações (inteiros, decimais, textos, booleanos).

int, float, str, bool

Operadores

Símbolos para realizar operações matemáticas, comparativas ou lógicas.

# Aritméticos: +, -, * /
# Comparativos: >, <, ==
# Lógicos: and, or, not

Condicionais

Permitem que o programa tome decisões com base em condições.

if temperatura > 25:
    print("Está quente!")
else:
    print("Temperatura agradável.")

Repetição

Permitem executar um bloco de código várias vezes (loops).

# Loop 'for'
for i in range(3):
    print("Repetindo...")

# Loop 'while'
contador = 0
while contador < 3:
    contador += 1

Funções

Blocos de código reutilizáveis que realizam uma tarefa específica.

def somar(a, b):
    return a + b

resultado = somar(5, 3)

Estruturas de Dados

Formas de organizar e armazenar coleções de dados.

# Lista (mutável, ordenada)
minha_lista = [1, 2, "texto"]

# Tupla (imutável, ordenada)
minha_tupla = (1, 2, "texto")

# Dicionário (chave-valor, não ordenada, mutável)
meu_dicionario = {"nome": "João", "idade": 30}

# Conjunto (elementos únicos, não ordenado)
meu_conjunto = {1, 2, 2, 3} # {1, 2, 3}

A Ferramenta Essencial: Pandas

Com a lógica em mente, apresentamos o Pandas, a biblioteca mais popular para manipulação e análise de dados em Python. Ela nos dá superpoderes para ler, transformar, limpar e analisar dados tabulares (como planilhas) de forma intuitiva e eficiente.

Estruturas de Dados

O Pandas se baseia em duas estruturas principais:

Series

Uma coluna de dados, como uma lista com rótulos (índices).

import pandas as pd
s = pd.Series([10, 20, 30], name="Minha Série")
# Saída:
# 0    10
# 1    20
# 2    30
# Name: Minha Série, dtype: int64

DataFrame

Uma tabela completa, com linhas e colunas, como uma planilha.

import pandas as pd
data = {'Nome': ['Alice', 'Bob'],
        'Idade': [25, 30]}
df = pd.DataFrame(data)
# Saída:
#     Nome  Idade
# 0  Alice     25
# 1    Bob     30

Por que usar Pandas?

  • ✓

    Facilidade de Uso: Sintaxe simples para operações complexas.

  • ✓

    Eficiência: Otimizado para trabalhar com grandes volumes de dados.

  • ✓

    Flexibilidade: Lê dados de CSV, Excel, SQL e muito mais.

  • ✓

    Poder de Análise: Ferramentas integradas para agrupar, filtrar e limpar dados.

Operações Comuns com Pandas

O Pandas oferece uma vasta gama de funções para manipular e analisar seus dados. Aqui estão algumas das mais utilizadas:

Leitura de Dados (`read_csv`, `read_excel`)

Importa dados de diferentes formatos para um DataFrame.

# Lendo um arquivo CSV
df_csv = pd.read_csv('dados.csv')

# Lendo um arquivo Excel
df_excel = pd.read_excel('dados.xlsx')

Seleção de Dados (`[]`, `loc`, `iloc`)

Acessa colunas ou linhas específicas do DataFrame.

# Selecionar uma coluna
nomes = df['Nome']

# Selecionar uma linha pelo índice (posição)
primeira_linha_iloc = df.iloc[0]

# Selecionar uma linha pelo rótulo do índice
# (se o índice for um nome, por exemplo)
# primeira_linha_loc = df.loc['Alice']

Filtragem de Dados (`[]` com condição)

Seleciona linhas que atendem a uma ou mais condições.

# Filtrar pessoas com mais de 28 anos
df_maiores_28 = df[df['Idade'] > 28]

# Filtrar por múltiplas condições
df_condicao = df[(df['Idade'] > 25) & (df['Cidade'] == 'São Paulo')]

Agrupamento de Dados (`groupby`)

Agrupa dados por uma ou mais colunas e aplica funções de agregação.

# Média de idade por cidade
media_idade_cidade = df.groupby('Cidade')['Idade'].mean()

Limpeza de Dados (`dropna`, `fillna`)

Trata valores ausentes (NaN) ou inconsistências.

# Remover linhas com valores ausentes
df_limpo = df.dropna()

# Preencher valores ausentes com um valor específico
df_preenchido = df.fillna(0)

# Preencher valores ausentes com a média da coluna
# df_media = df['Idade'].fillna(df['Idade'].mean())

Criação de Novas Colunas

Adiciona novas colunas baseadas em cálculos ou outras colunas.

# Criar uma coluna 'Idade_em_anos'
df['Idade_em_anos'] = df['Idade']

# Criar uma coluna 'Status' baseado na idade
df['Status'] = ['Adulto' if idade >= 18 else 'Jovem' for idade in df['Idade']]

Estatísticas Descritivas (`describe`, `info`)

Obtém um resumo estatístico ou informações sobre o DataFrame.

# Resumo estatístico das colunas numéricas
resumo_estatistico = df.describe()

# Informações sobre o DataFrame (tipos de dados, não nulos)
df.info()

Junção/Combinação de DataFrames (`merge`, `concat`)

Combina DataFrames com base em colunas comuns ou empilhando-os.

# Exemplo de merge (junção de tabelas)
# df_total = pd.merge(df1, df2, on='ID_Comum')

# Exemplo de concat (empilhar tabelas)
# df_combinado = pd.concat([df1, df2])

Visualização de Dados: Matplotlib

Matplotlib é uma biblioteca de plotagem 2D para Python que gera figuras de qualidade de publicação em uma variedade de formatos de hardcopy e ambientes interativos em todas as plataformas. É a base para muitas outras bibliotecas de visualização e é amplamente utilizada para criar gráficos estáticos e dinâmicos.

Por que Matplotlib com Pandas?

Pandas é excelente para manipular e preparar seus dados. Matplotlib, por sua vez, é a ferramenta perfeita para transformar esses dados preparados em visualizações significativas. Juntos, eles formam um pipeline poderoso para análise e apresentação de dados.

  • ✓

    Preparação de Dados: Pandas organiza e limpa os dados.

  • ✓

    Visualização Flexível: Matplotlib permite personalizar cada aspecto do gráfico.

  • ✓

    Tipos de Gráficos: Crie desde gráficos de linha simples a gráficos 3D complexos.

Projeto Final: Análise de Consumo de Energia Urbana

Neste projeto final, aplicaremos os conhecimentos de Pandas e Matplotlib para analisar dados de consumo de energia em uma cidade. Este tipo de análise é fundamental para diversas engenharias, como a Elétrica (otimização de rede), Civil (planejamento urbano e infraestrutura) e Ambiental (sustentabilidade e impacto). Vamos simular um dataset de consumo mensal por setor.

Passo 1: Carregamento e Inspeção dos Dados

Primeiro, carregamos os dados em um DataFrame Pandas e fazemos uma inspeção inicial para entender sua estrutura.

import pandas as pd

# Dados simulados de consumo de energia (em GWh)
data = {
    'Mes': ['Jan', 'Fev', 'Mar', 'Abr', 'Mai', 'Jun'],
    'Residencial': [100, 110, 105, 120, 115, 130],
    'Industrial': [80, 85, 90, 88, 92, 95],
    'Comercial': [60, 65, 62, 70, 68, 75]
}
df_energia = pd.DataFrame(data)

# Visualizar as primeiras linhas
# print(df_energia.head())

# Obter informações sobre o DataFrame
# print(df_energia.info())

Passo 2: Análise e Transformação de Dados

Vamos calcular o consumo total de energia por mês e identificar o setor com maior consumo em cada mês.

# Calcular o consumo total mensal
df_energia['Total'] = df_energia['Residencial'] + df_energia['Industrial'] + df_energia['Comercial']

# Encontrar o setor de maior consumo por mês
def setor_maior_consumo(row):
    setores = ['Residencial', 'Industrial', 'Comercial']
    max_consumo = 0
    setor = ''
    for s in setores:
        if row[s] > max_consumo:
            max_consumo = row[s]
            setor = s
    return setor

df_energia['Setor_Dominante'] = df_energia.apply(setor_maior_consumo, axis=1)

# Visualizar o DataFrame com as novas colunas
# print(df_energia)
